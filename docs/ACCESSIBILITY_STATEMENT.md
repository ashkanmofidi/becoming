# Accessibility Statement

**Becoming.. Focus Timer**
**Last updated:** April 9, 2026

## Commitment

Becoming.. is committed to ensuring digital accessibility for all users. We continually improve the user experience for everyone and apply relevant accessibility standards.

## Conformance Status

Becoming.. partially conforms with WCAG 2.1 Level AA. "Partially conforms" means that some parts of the content do not fully conform to the accessibility standard.

## Accessibility Features

- **Keyboard navigation**: All interactive elements accessible via keyboard
- **Screen reader support**: ARIA live regions for timer state changes, configurable verbosity (Minimal/Standard/Verbose)
- **Skip to content**: Skip navigation link for keyboard users
- **Color blind modes**: Protanopia, Deuteranopia, Tritanopia support with color palette adjustments
- **High contrast mode**: WCAG AAA (7:1) contrast ratios, thick focus indicators
- **Large tap targets**: 56px minimum touch targets (optional, in Accessibility settings)
- **Reduced motion**: Disables all animations when enabled or when OS prefers-reduced-motion
- **Font scaling**: 80% to 150% font size options
- **Dark and light themes**: Both designed with WCAG AA contrast ratios

## Known Limitations

- Charts on the Dashboard page may not be fully accessible to screen readers
- Some dynamic content updates may not be immediately announced
- Timer countdown is visual — screen reader announces at intervals, not every second
